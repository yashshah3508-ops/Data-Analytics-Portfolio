District Development Analysis (Excel Dashboard)

Overview

This project analyzes district-level data to evaluate development performance using key indicators such as literacy, healthcare, and education infrastructure.

Dataset

The dataset includes:

- State and District
- Literacy Rate (%)
- Population
- Health Centers
- Schools

Methodology

- Cleaned and structured raw data using Excel
- Created calculated metrics such as:
  - Health Centers per 1000 population
  - Schools per 1000 population
- Built a composite Development Score using weighted indicators
- Used Pivot Tables to summarize data by state and district
- Developed interactive dashboards using charts and slicers

Dashboard Features

- Average Development Score by State
- Top Performing Districts
- District-wise comparison using slicers
- Executive summary dashboard with key KPIs

Key Insights

- State B shows higher average development compared to State A
- Certain districts significantly outperform others due to better infrastructure
- Development varies widely even within the same state

Tools Used

- Microsoft Excel
- Pivot Tables
- Data Visualization (Charts & Slicers)
